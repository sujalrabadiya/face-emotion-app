# 😊 Real-Time Face Emotion Detector

This is a simple web-based face emotion detection application built with **Streamlit**, **OpenCV**, and a **Keras deep learning model**. It allows users to upload an image and instantly see the predicted facial emotion.

## 🔍 Features

- Upload any face image (JPG, JPEG, PNG)
- Detects emotions such as:
  - Angry
  - Disgust
  - Fear
  - Happy
  - Sad
  - Surprise
  - Neutral
- Highlights the face and shows the predicted emotion on the image
- Clean and responsive UI with **Streamlit**
